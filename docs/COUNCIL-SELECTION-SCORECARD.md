# Council Selection Scorecard

Use this scorecard to choose the first three Queensland councils for pilot ingestion.

## Criteria

- public planning scheme accessibility
- building forms and checklist availability
- GIS and zoning data accessibility
- permit guidance clarity
- historical decision visibility
- source freshness and update traceability
- expected market density for target permit classes

## Scoring Model

Score each dimension from 1 to 5.

- 1: poor or inaccessible
- 3: workable with manual effort
- 5: strong candidate for early automation

## Decision Rule

Select councils that combine:

- high source accessibility
- low ambiguity in small residential structure guidance
- enough permit demand to support early pilot learning

## Required Output

For each candidate council, record:

- council name
- state
- source URLs
- source types
- known gaps
- score by criterion
- overall recommendation
